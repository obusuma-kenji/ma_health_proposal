健康経営への取組み度と社員の健康度が、
M&Aにおける企業価値の要素となる

2025年１０月２日

一般社団法人組織診断協会　代表理事　江田幸央　様　

1. 提案の目的

M&A市場における新たなサービス領域として、「労務承継部門による健康経営デューデリジェンス（健康経営DD）」を事業化し、社労士の専門性を活かした差別化サービスを確立することを提案いたします。

2. 提案の背景

2.1 市場環境の変化

（1）人的資本経営への注目の高まり

  • 2023年3月期より有価証券報告書における人的資本情報開示が義務化
  • 人材版伊藤レポート2.0において「健康経営への投資とWell-beingの視点」が明記
  • 健康経営は人的資本経営の土台として位置づけられている

（2）投資家の評価基準の変化

  • 米国では2020年に人的資本情報の開示が義務化
  • 東京証券取引所が企業統治指針を改訂し、人的資本開示を強化
  • 投資家は財務諸表に加え、健康経営等の人的資本指標で企業価値を判断

（3）M&A市場における課題

  • 従来の労務DDは未払残業代、労使トラブル等のリスク発見が中心
  • 健康経営を体系的に評価する手法が確立されていない（市場機会）
  • 人的資本の質的評価への需要が高まっている

3. 提案内容

3.1 サービス概要

サービス名: 「健康経営デューデリジェンス（健康経営DD）」

サービス内容:
M&Aにおいて、対象企業の健康経営への取り組み度と従業員の健康度合を評価し、企業価値への影響を定量化・可視化する専門サービス

3.2 提供価値

（1）買収側企業への価値

  • 隠れた労務リスク（健康データ、メンタルヘルス、過重労働）の発見
  • プレゼンティーイズムコスト（生産性損失）の定量的把握
  • 買収後の統合コスト・離職リスクの予測

（2）売却側企業への価値

  • 健康経営投資の可視化による企業価値向上
  • 人的資本情報の適切な開示支援
  • 買収交渉における優位性確保

（3）投資家・ステークホルダーへの価値

  • 人的資本の質的評価に基づく投資判断材料の提供
  • ESG投資の観点からの企業価値評価支援

3.3 評価項目（チェックリスト）

定量評価項目

  • 健康診断受診率・有所見率
  • ストレスチェック実施状況と高ストレス者割合
  • 長時間労働の実態（月45時間超過者数、80時間超過者数）
  • 離職率（全体・若年層）
  • 労災発生件数・内容
  • 健康投資額（1人当たり）
  • アブセンティーイズム・プレゼンティーイズムコスト

定性評価項目

  • メンタルヘルス対策の体制・実効性
  • 健康経営戦略の経営への組み込み度
  • 産業保健体制（産業医、保健師の配置状況）
  • 健康経営優良法人認定の有無
  • ワークライフバランス施策の実施状況

3.4 成果物

  1. 健康経営DD報告書（リスク評価、改善提案含む）
  2. 健康経営スコアカード（業界ベンチマーク比較）
  3. 企業価値への影響試算書（生産性損失額、潜在的債務等）
  4. PMI健康経営ロードマップ（統合後の施策提案）

4. 差別化要素と競争優位性

4.1 市場における優位性

要素 当社の強み
未開拓市場 健康経営DDは新領域で競合が極めて少ない
専門性 社労士の労務・社会保険・安全衛生の知識を完全活用
時代適合性 人的資本開示義務化という追い風
コスト競争力 弁護士（500-1000万円）比で優位な価格設定が可能

4.2 既存サービスとの相乗効果

  • 既存の労務顧問先へのアップセル機会
  • M&A仲介会社との連携による新規顧客開拓
  • IPO支援サービスへの横展開可能性

5. 実施計画

フェーズ1：準備期間（3ヶ月）

実施事項:

  1. 評価手法・指標の標準化
  2. 評価ツール・テンプレート開発
  3. 業界別ベンチマークデータの収集開始
  4. M&Aアドバイザー、会計士との連携体制構築

目標成果:

  • 健康経営DDマニュアル完成
  • 評価ツール（Excelベース）完成

フェーズ2：パイロット期間（6ヶ月）

実施事項:

  1. 中小企業M&A案件で2-3件のパイロット実施
  2. サービス内容のブラッシュアップ
  3. 事例・実績の蓄積と可視化
  4. M&A仲介会社への営業活動

目標成果:

  • パイロット案件3件完了
  • 成功事例1件以上確保

フェーズ3：本格展開期間（6ヶ月以降）

実施事項:

  1. サービスの正式リリース
  2. 業界団体・セミナーでの発信
  3. ベンチマークデータベースの拡充
  4. 専門チームの組成

目標成果:

  • 年間10件以上の受注
  • 業界内での認知度確立

6. リスクと対策

6.1 想定されるリスク

リスク 対策
市場認知度不足 M&A仲介会社との提携、セミナー開催による啓蒙活動
評価基準の信頼性 学術機関・業界団体との協力、エビデンスベースの手法確立
競合参入 早期の実績蓄積、ノウハウの体系化によるブランド確立
人材不足 外部専門家（産業医等）とのネットワーク構築

6.2 成功のための重要要素

  1. M&A仲介会社との強固な連携関係の構築
  2. 評価の客観性・再現性の確保
  3. 成功事例の早期確保と積極的な発信
  4. 継続的な評価手法のアップデート

7. 期待される効果

7.1 定量的効果

  • 既存顧客の単価向上: 労務顧問先へのクロスセル
  • 市場シェア獲得: 健康経営DD市場におけるファーストムーバー

7.2 定性的効果

  • ブランド価値向上: 先進的な社労士事務所としての認知
  • 専門性の強化: 人的資本経営分野での専門家ポジション確立
  • 社会的意義: M&A市場における人的資本評価の質向上への貢献
  • 業界への貢献: 社労士の新たな役割創出

8. 結論

本提案は、以下の理由から早急な事業化を推奨いたします。

1. 市場環境が追い風: 人的資本開示義務化、ESG投資拡大
2. 競合優位性: 未開拓領域で先行者利益を獲得可能
3. 専門性の適合: 社労士の強みを最大限活用
4. 事業性: 初期投資に対し、十分なリターンが見込める
5. 社会的意義: M&A市場の健全化、企業の持続的成長に貢献

健康経営とM&Aを結びつけた本サービスは、社労士業界に新たな付加価値をもたらし、当事務所の競争力を大きく向上させる戦略的施策であると確信しております。
